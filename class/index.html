Navegador web

Un navegador web (en inglés, web browser) es un software, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad basica de navegador web es permitir la visualización de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari: Safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e iOS

Mozilla FIrefox es un navegador web libre y de código abierto desarrollado para LInux, Android, iOS, macOS y Microsoft Windows coordinado por la Corporación Mozilla y La Fundación Mozilla.

Google Chrome: Google Chrome es un navegador web de código cerrado desarrolaldo por Google, aunque derivado por proyectos de código abierto.
